Day 1: Script Writing Checklist (Retention Engine)

Your goal: Write a script that retains attention from start to finish.

Quick Table Overview

Step Task Summary Time Estimate
1 Hook Writing 20–30 mins
2 Structure Planning (AIDA/Story Arc) 15–20 mins
3 Script Writing (Bullet or Word-by-Word) 60–90 mins
4 Retention Boost Techniques 30 mins
5 Final Polish & Reading Aloud 15–30 mins

1. Magnetic Hook

  • Start with a bold question, statement, or shock value
  • Use emotion: Fear, Curiosity, Desire, Surprise
  • Set a clear promise: “In this video, you’ll get ____”
  • Tease end result or plot twist

2. Structure the Core (AIDA or Hero’s Journey)

  • AIDA: Attention > Interest > Desire > Action
  • Hero’s Journey: Problem > Struggle > Breakthrough > Victory
  • Create 3–5 sub-points that deliver constant value

3. Script It Out

  • Choose format: Bullet points or Full script
  • Add hooks every 30–45 secs
  • Include personal stories and examples
  • Write notes for camera/B-roll/memes

4. Boost Viewer Retention

  • Use pattern interrupts: Zooms, pops, memes
  • Add open loops and cliffhangers
  • Ask direct questions or give mini challenges
  • Use humor or surprises to create emotion spikes

5. Polish the Script

  • Read the script aloud for flow
  • Adjust pacing and clarity
  • Highlight key phrases or delivery points
  • Ensure it works for short attention spans

Pro Tip:

Your script is your map. Even the best editing won’t fix a boring story. Make it a performance on paper first.
